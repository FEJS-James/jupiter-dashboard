📖 Manual Testing Guide

⚠️ Important: Navigate to http://localhost:3000/tasks before testing!

1. Global Navigation Shortcuts

  • ? - Opens help modal with searchable shortcuts
  • Ctrl+K - Opens command palette
  • g + d - Navigate to Dashboard
  • g + b - Navigate to Board/Tasks
  • g + p - Navigate to Projects
  • g + a - Navigate to Analytics
  • g + n - Navigate to Notifications

2. Kanban Board Navigation

  • j/k - Navigate down/up through tasks
  • h/l - Navigate left/right between columns
  • Enter - Open selected task details
  • 1-6 - Switch between columns

3. Task Management

  • n - Create new task
  • e - Edit selected task
  • d - Delete selected task
  • f - Focus search input
  • m - Move task to next status
  • Ctrl+D - Duplicate task

4. Advanced Features

  • Ctrl+T - Toggle theme
  • Escape - Cancel/close modals, clear selections
  • Shift+←/→ - Move task to previous/next status

✅ Expected Behaviors

  • Visual feedback when tasks are selected (highlighting/rings)
  • Help modal is searchable and well-organized
  • Command palette provides quick access to actions
  • Shortcuts respect input focus (don't interfere with typing)
  • Accessibility: proper focus management and ARIA labels
  • Performance: no lag when using shortcuts rapidly
